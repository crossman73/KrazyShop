# 📱 KrazyShopn MVP 개발 계획 (3주)

## 개발 원칙
- **Daily Demo**: 매일 동작하는 기능 구현
- **User-Centric**: 사용자 관점에서 기능 우선순위
- **Investor-Ready**: 투자자 데모 가능한 수준
- **무료 환경**: 비용 제로 유지

## Week 1: Foundation & Core (기반 구축)

### Day 1-2: 프로젝트 설정
**목표**: 개발 환경 완전 구축
```
✅ Supabase 프로젝트 생성 + DB 스키마
✅ Next.js 프로젝트 초기화 + Vercel 배포
✅ 기본 UI/UX 컴포넌트 (TailwindCSS)
✅ 사용자 인증 (Supabase Auth)
```
**데모**: 로그인/회원가입 동작

### Day 3-4: 이미지 업로드 & OCR
**목표**: 핵심 기능 프로토타입
```
✅ 이미지 업로드 (Supabase Storage)
✅ Tesseract.js OCR 구현
✅ 기본 텍스트 추출 UI
✅ 에러 핸들링 & 로딩 상태
```
**데모**: 화장품 이미지 → 텍스트 추출

### Day 5-7: 기본 성분 분석
**목표**: 성분 분석 로직 구현
```
✅ 성분 데이터베이스 초기 구축 (100개 성분)
✅ 성분 매칭 알고리즘
✅ 안전성 점수 계산
✅ 분석 결과 UI
```
**데모**: 성분 텍스트 → 안전성 분석

### Day 11-12: 제품 비교 & 추천


### Day 13-14: UX 최적화
**목표**: 사용자 경험 개선
```

```
